Подготовка к работе

Обновлять ASystem Touch вручную, чтобы не пропустить заказ

Отметить начало задания в Asystem Touch

Откопировать .pdf-макет и .jpg-превью в рабочую директорию:

WORK/[год]-[номер заказа]/ Например, 26-012345
26K-012345

Cборная упаковка, несколько деталей:

WORK/[год]-[номер заказа]s/[год]-[номер заказа]/01[...]04/

Например, 26-012345s

26-012345s/26-012345-01/

26-012345s/26-012345-02/

26-012345s/26-012345-03/

Проверить техзадание на предмет комментариев. Они могут быть также в рабочей директории в .txt-формате. Комментарии в техзадании имеют приоритет над обычным ходом работ

При явном несоответствии комментариев техзаданию спрашивать о происходящем технологов

Вообще надо листать описания из тача вдоль и поперёк

Preflight: автопроверка

При наличии ошибок вернуть макет дизайнерам с описанием проблемы

Проверить цветовые каналы:

  • Cut в overprint
  • Varnish в overprint (инверсный)
  • Количество красок равно указанному в техзадании

    При несовпадении количества каналов печати с техзаданием писать о проблеме технологу

Во всём остальном можно довериться автопроверке в Preflight

О любых ошибках сообщать и возвращать дизайнерам на доработку

Prinect Trap Editor: создать автотреппинг 0_06

Если есть два прогона с пересекающимися красками, то величина треппинга увеличивается до 0_1

Переименовать файл с треппингом:

[год]-[номер]_TRAP_[прежнее название файла].pdf

Автотреппинг выдал создание 0 объектов / Треппинг не требуется:

[год]-[номер]_WOTRAP_[прежнее название файла].pdf

P.S. Если треппинг по времени затягивается (более 10-15 минут), значит что-то не в порядке с файлом: прозрачности, куча векторных объектов и т.д. Такой файл вернуть файл дизайнерам

Приготовление макета упаковки по шаблону штампа

Искать самый свежий шаблон штампа:

  • Если в задании указан картон: Templates/КАРТОН
  • Если в задании указана бумага: Templates/офсет

Рекомендуется смотреть самый актуальный штамп по дате изменения файла

Предыдущие версии необходимо проверить на предмет соответствия последним требованиям подготовки к печати

При наличии нескольких версий спуска одного штампа удали более старые версии

Cоздать шаблон спуска в Signa через New Job

При отсутствии шаблона спуска для штампа

В папку Templates создать New Job stamp [номер штампа]

  1. —
  2. Mode: Packaging

  3. Plate Template XL106

    Проставить размер листа бумаги, клапан Y=13mm

  4. Найти штамп по номеру в формате .cf2 в директории. При отсутствии оного создать папку под номер штампа и перетащить туда .cf2 финальной версии /штампы/№/аут/[...].cf2

    Если штампа в формате .cf2 нет, писать об этом технологу. Если есть но не в cf2, писать конструктору. Искать кто делал, просить написать нужный формат.

  • Выделить весь штамп, проставить в Cutting
  • Нажать Generate Bleed для генерации припусков 2.5mm

    Выделиться заливкой должны все детали конструкции. Если какой-то из контуров разомкнут, писать об этом конструктору

  • Разрешить конфликты созданных вылетов при помощи инструмента ClipPath Conflict Resolution
  • Клапан вырубки Y=13
  • Если изделий несколько видов на одном листе, снять флажок «Same Content for All 1ups of a Shape»
  • Проверить привязку всех ключевых типографских меток по Subject, добавить недостающие метки

Перед созданием спуска из нового шаблона необходимо отправить директорию с ImposePDF Анастасии Офицеровой на согласование

Настроить порядок цветов и настройки внутри палитры Colors:

  • Чёрный / самый тёмный Pantone
  • Триадные CMY
  • Прочие пантоны
  • Сетка U, дубликат канала печати, обычно в 30-70%
  • Цвет под сеткой C обычно это пантон; но может быть и что-то из триады
  • Varnish. Проверить тип печати Usage: Coating Unit. Иначе лак попадёт на цветовые шкалы. Автоматически помечается как L1
  • Cut. Проверить тип печати Usage: Tool, Type: DieLine. Иначе контур штампа попадёт в печать по полной программе

Если печатный макет из пантонов

При печати из пантонов проставляем метки цвета вручную,
В меню Assignment of job colors to mark colors выбираем радиобаттон User-defined:
Проставляем флажок Allow spot colors for BCMY

Проверить все необходимые метки

С привязкой к листу:

  • Японские кресты приводки Ars/cross BG 4 шт c каждого угла
  • RegisterLine
  • Метка верного угла правый нижний угол
  • Метка наброса
  • Цветовые шкалы

    Для машины XL-106 Dipco_actual

    Для машины XL-105 Dipco_XL-105

    XL-106, свыше 8 цветов печати microscale/F105_Micro-10

    XL-105, свыше 2 пантонов scales/old_Image control/6S+

    Шкалы должны отступать от правого и левого края 3-5mm

    Можно обрезать или перекрыть буквы, обязательны только квадраты

  • Краскосъём Только если есть 5mm на каждую полоску, иначе не надо
  • Краскосъём серый
    1. Выделить всю область до ножей
    2. Поставить флажок overlay, все цвета наложатся друг на друга
    3. Проверить наличие всех красок печати, каждую выставить в 40%
    4. Выставить позиционирование метки по Subject
    5. Отвязать пропорциональность ширины с высотой, увеличить метку на 5мм по каждой стороне
    6. Позиционировать метку по припускам
  • Приводные метки для вырубки CutMark-3mmПометить штампы в нижней части спуска со всех сторон вручную

  • Оптическая метка mark-optical

    справа снизу; лучше и справа, и слева. И по центру, если поместится

    Y=136 при клапане 13 mm

    Важно, чтобы расстояние от края листа до чёрной области метки было всегда 151mm
  • Формат бумаги Ars/PaperFormat Если пересекается с mark-optical, сдвинуть выше
  • Ограничительные линии для выборочного вд-лака по 2 шт Laque/RegisterLine_Horizontal Laque/RegisterLine_Vertical

    создать рамку по припуску под обрез

  • Надпись о клапане Ars/Clapan 13 mm в левый нижний угол

  • Нумерация изделий Bobst Marks/1... пронумеровать изделия в ряд слева направо начиная с нижнего левого угла
  • Марка зрительного контроля для нескольких видов изделий

    разместить на контур штампа, на каждое изделие свой цвет печати

Подготовка можнтажной плёнки(VD_progon1, foil, kongrev)

Редактировать линии лака:

  1. Сменить позиционирование на Paper
  2. Выставить координаты на 0; линии окажутся по краям листа
  3. Сменить цвет с лака на цвет Black

Убрать:

  • Все краскосъёмы
  • Все цветовые шкалы

Редактировать [номер заказа]/Sheets/ImposePDF_paper в Acrobat, PDF Toolbox

  1. Откопировать в директорию для заказа монтажек
  2. Вкладка слева Spot Colors
  3. И вырубку, и печать, все каналы проставить в Black в столбце Target, сохранить файл
  4. Прислать директорию файла монтажки Александру

Подготовка флексоформы

Убираем метки:

  • Цветовые шкалы
  • Краскосъёмы
  • Надпись о клапане Ars/Clapan 13
  • Ограничительные линии по лаку

Добавить метку:

KLM/krai_lista привязать по бумаге по центру снизу
  1. Экспортировать impose pdf_paper в папку для заказа флексоформ
  2. Редактировать pdf через PDF Toolbox, вкладка слева Geometry Control
  3. В столбец Width дописать -0,4
  4. В столбец Height восстановить значение из техзадания. Теперь макет непропорционально сжат на 0,4мм
  5. С вкладки слева Spot Colors все цветовые каналы переназначаем в Black
  6. Прислать директорию с файлом технологам
